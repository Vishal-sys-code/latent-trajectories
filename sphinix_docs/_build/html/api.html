Core API Reference¶

This section documents the core Python modules underpinning the analytical pipeline. All internal components rely on PyTorch for high-dimensional computations and NumPy/Pandas for downstream statistics and visualizations.

Metrics Module¶

src.metrics.compute_convergence_matrix(trajectories: List[HiddenStateTrajectory]) → ndarray[source]¶

Computes a pairwise convergence matrix for a list of trajectories. Positive values indicate convergence, negative values indicate divergence.

D_ij(l) = || T_i(l) - T_j(l) || C_ij = D_ij(1) - D_ij(L)

Parameters:

trajectories – A list of HiddenStateTrajectory objects.

Returns:

np.ndarray of shape [num_prompts, num_prompts] containing pairwise convergence scores.

src.metrics.compute_convergence_score(trajectories: List[HiddenStateTrajectory], labels: List[str]) → ndarray[source]¶

Computes the convergence score for each layer: Convergence Score(l) = D_between(l) - D_within(l)

where D_between is the mean pairwise distance between trajectories from different categories, and D_within is the mean pairwise distance between trajectories from the same category. Higher values imply stronger category separation, lower values imply convergence.

Parameters:

  • trajectories – A list of HiddenStateTrajectory objects.

  • labels – A list of category labels for each prompt.

Returns:

np.ndarray of shape [L] containing the convergence score at each layer.

src.metrics.compute_curvature(trajectories: List[HiddenStateTrajectory]) → ndarray[source]¶

Computes the average curvature of the trajectory for each prompt. Curvature quantifies how much the trajectory bends during processing.

v_l = h_l - h_(l-1) v_(l+1) = h_(l+1) - h_l κ_l = arccos( (v_l · v_(l+1)) / (||v_l|| ||v_(l+1)||) ) κ(T) = (1/(L-2)) Σ κ_l

Parameters:

trajectories – A list of HiddenStateTrajectory objects.

Returns:

np.ndarray of shape [num_prompts] containing the overall trajectory curvature.

src.metrics.compute_layer_velocity(trajectories: List[HiddenStateTrajectory]) → ndarray[source]¶

Computes the average velocity at each layer transition across all prompts. Velocity measures the amount of movement between consecutive layers.

v_l = || h_(l+1) - h_l ||

Parameters:

trajectories – A list of HiddenStateTrajectory objects.

Returns:

np.ndarray of shape [L-1] containing the mean velocity at each layer transition.

Returns:

np.ndarray of shape [L] containing the silhouette score for each layer.

src.metrics.compute_per_prompt_convergence_score(trajectories: List[HiddenStateTrajectory], labels: List[str]) → ndarray[source]¶

Computes the convergence score for each prompt at each layer. Convergence Score_i(l) = mean(D_between_i(l)) - mean(D_within_i(l))

For a given prompt i: - mean(D_within_i) is the average distance to other prompts in the SAME category - mean(D_between_i) is the average distance to prompts in DIFFERENT categories

Parameters:

  • trajectories – A list of HiddenStateTrajectory objects.

  • labels – A list of category labels for each prompt.

Returns:

np.ndarray of shape [num_prompts, L] containing the convergence score.

src.metrics.compute_rsa_matrix(trajectories: List[HiddenStateTrajectory]) → ndarray[source]¶

Performs Representational Similarity Analysis (RSA) by computing the Spearman rank correlation between the Representational Dissimilarity Matrix (RDM) of each pair of layers.

Parameters:

trajectories – A list of HiddenStateTrajectory objects.

Returns:

np.ndarray of shape [L, L] containing the RSA matrix.

src.metrics.compute_trajectory_length(trajectories: List[HiddenStateTrajectory], normalized: bool = True) → ndarray[source]¶

Computes the total length of the trajectory for each prompt. Trajectory length measures how far a representation travels through latent space as it passes through the transformer.

If normalized=True, calculates: L_norm(T) = Σ || normalize(h_(l+1)) - normalize(h_l) ||_2

If normalized=False, calculates: L_raw(T) = Σ || h_(l+1) - h_l ||_2

Parameters:

  • trajectories – A list of HiddenStateTrajectory objects.

  • normalized – Whether to use L2-normalized hidden states.

Returns:

np.ndarray of shape [num_prompts] containing the trajectory lengths.

src.stats.cohens_d(group_a: ndarray, group_b: ndarray) → float[source]¶

Computes Cohen’s d effect size for two independent groups.

d = (mean(A) - mean(B)) / pooled_sd

Parameters:

  • group_a – A 1D numpy array of values for group A.

  • group_b – A 1D numpy array of values for group B.

Returns:

The Cohen’s d effect size.

Comprehensive comparison between two distributions. Returns permutation test p-value, Mann-Whitney U p-value, and Cohen’s D.

src.stats.permutation_test(group_a: ndarray, group_b: ndarray, num_permutations: int = 10000, random_seed: int | None = None) → float[source]¶

Performs a non-parametric permutation test to determine if the difference in means between two independent groups is statistically significant.

H0: group_a and group_b have the same mean. H1: group_a and group_b have different means.

Parameters:

  • group_a – A 1D numpy array of values for group A.

  • group_b – A 1D numpy array of values for group B.

  • num_permutations – Number of permutations to run.

  • random_seed – Optional seed for reproducibility.

Returns:

The two-sided p-value.
